# SoundFor1C_UT11_5
Звуковое сопровождение операций для УТ 11.5 (или любой другой конфигурации) <br>
На примере УТ 11.5 озвучка печати чека. <br>
Проблема: <br> 
При использовании отслеживаемых товаров чек печатается долго <br>
<br>
Решение: <br>
Использовать голосовое оповещение для удержания покупателя до окончания печати чека <br>
<br>
Возможно переделать для любой конфигурации 1С <br>
<br>
Внимание, образец звука в архиве <br>
<br>
<br>
1. Для сборки расширения создайте пустое расширение <br>
2. Скачайте git clone https://github.com/rf-bessome/SoundFor1C_UT11_5 <br>
3. Загрузите в пустое расширение файлы, которые находятся в архиве в конфигураторе<br>
через меню конифгурация - загрузить из файлов в окне расширения <br>
<br>
